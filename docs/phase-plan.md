# Rattracker Build Phases

## Phase 1 — Core Foundation

Build only the systems that make the product feel real and stable:

1. Luxury responsive app shell with sidebar, topbar, dashboard layout, and dark mode.
2. Rat profile system with image, name, DOB, sex, coat, temperament, lineage, weight, notes, and pill-style tags.
3. Dashboard homepage with total rats, recent litters, alerts, weight trends, and quick actions.
4. Tasks and reminders for weighing, cage cleaning, medication, and pairing dates.
5. Simple Supabase-ready database tables: `rats`, `litters`, and `tasks`.

## Phase 2 — Breeder Operations

Add pedigree previews, health logs, litter detail pages, breeding planner, inventory, and finances.

## Phase 3 — Advanced Visuals

Add interactive pedigree trees, health heatmaps, drag-and-drop pairings, and richer charts.

## Phase 4 — AI Systems

Only after enough data exists, add pairing recommendations, anomaly detection, genetic predictions, and smart alerts.
